## 🌟 주요 기능
- 각 대학교에 대한 실생활 정보 제공
  - 맛집
  - 평균 월세
  - 주변 놀거리
  - 축제 라인업
- 학교 기본 정보 제공
  - 경쟁률
  - 등록금
  - 취업률
  - 학과 정보
- 학과 추천 설문: "나에게 맞는 학과는?"
- 입시 일정 달력 및 알림 기능
- 관심 대학교 즐겨찾기 기능
